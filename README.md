## 📱 Installation Mobile

1. **Ouvrir le site** sur votre téléphone
2. **Chrome/Safari** vous proposera "Ajouter à l'écran d'accueil"
3. **Cliquer sur "Ajouter"**
4. L'app apparaîtra sur votre écran d'accueil comme une vraie app !
